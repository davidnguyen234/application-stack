Building a Vue.js To-Do list web app on a MEVN stack

1. Follow the steps 1-5 from the MEVN tutorial to create a new project on GCP.

2. Click on the hamburger and click on Compute Engine and select ssh for the VM instance that was created.

3. Click on ssh. This will open up a terminal window.

Add Package Manager

4. We need to install Node Version Manager (NVM).

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.1/install.sh | bash

*Close your terminal and relaunch it before moving on to the next step

Adding Node.js

5. Installing the latest version of Node.js run the following:

nvm install stable

To verify that NVM has been installed run the following:

nvm --version

Install Git

6. Next we installed git: Run the following

sudo apt install git

7. You will be asked if you want to continue. Enter 'y' and press enter.

8. Next we associated our git user.name with this installation:

git config --global user.name "FirstName LastName"

9. Next we associated our git user.email with this installation:

git config --global user.email "youremail@email.com"

Clone repo

10. Run the following:

sudo git clone https://github.com/generocha/mevn-todo.git

Start the Vue CLI

11. Run the following:

cd ../ npm run serve

12. Open a second ssh window

13. Run the following:

cd mevn-todo/server npm run dev

14. Open app in the browser. http://<'External IP'>:8080
